Key Features of HTML5

1. Multimedia Support

HTML5 provides native support for audio and video elements, allowing easy embedding of media without the need for third-party plugins.

2. Canvas for Drawing

The <canvas> element allows developers to draw 2D graphics directly in the browser using JavaScript.

3. Geolocation API

HTML5 introduces the Geolocation API, allowing websites to access and display the user's geographical location with their consent.
